# AI 做歌系统

给一个灵感，AI 帮你写词、作曲、生成短视频。

## 你能用它做什么？

- 🎵 **写歌** — 给一句灵感（比如「深夜一个人走在路上」），出一首完整的歌
- 🔄 **翻唱** — 上传一首歌，换一种风格重新唱（比如把流行歌变成 R&B 版）
- 🎬 **做视频** — 给歌配上动态背景 + 卡点歌词，直接发抖音
- 📝 **写歌词** — 只要歌词不要曲，也行

## 怎么开始？

### 第 1 步：下载项目

让分享给你链接的人帮你下载，或者：

1. 打开终端（Mac 上按 `Cmd + 空格`，输入「终端」，回车）
2. 复制粘贴下面这行，回车：

```
cd ~/Desktop && git clone https://github.com/xuejin462-a11y/-AI-.git && cd -AI-
```

桌面上会出现一个 `-AI-` 文件夹，这就是项目。

### 第 2 步：安装（一次就好）

在终端里输入：

```
cd ~/Desktop/-AI- && bash setup.sh
```

安装向导会一步步引导你：
- 自动安装需要的软件
- **一个一个问你要 API 密钥**，每个都告诉你去哪里拿、怎么复制
- 最后自动测试是否配置成功

> 💡 不知道怎么获取 API 密钥？别担心，安装向导里每一步都有详细说明。
> 也可以看下面的「API 密钥获取教程」。

### 第 3 步：打开做歌界面

```
cd ~/Desktop/-AI- && python3 app.py
```

浏览器会自动打开一个网页，在上面操作就行。

> 以后每次想做歌，只需要重复第 3 步。

## 做歌界面长什么样？

打开后左边有 4 个页面：

| 页面 | 做什么 |
|------|--------|
| 🎵 写一首歌 | 填灵感/贴歌词/传参考曲 → 选风格 → 点按钮 → 出歌 |
| 🔄 二创翻唱 | 传原曲 → 选新风格 → 出歌 |
| 🎬 生成视频 | 传歌曲 → 选情绪模版 → 出抖音视频 |
| ⚙️ 设置 | 填/改 API 密钥（有详细教程） |

### 写歌的 3 种方式

**方式 A：我有一个灵感**
> 只需要一句话描述你的想法，比如：
> - 「深夜加班后走在空旷马路上」
> - 「异地恋，她在北京他在成都，高铁 8 小时」
> - 「像《晴天》那种感觉」

**方式 B：我已经写好歌词了**
> 直接粘贴歌词，可以选择上传一首参考曲让 AI 参考旋律

**方式 C：我有参考曲**
> 上传一首你喜欢的歌，AI 会参考它的旋律，生成一首全新的歌

### 可以调的参数

| 参数 | 什么意思 | 怎么选 |
|------|---------|--------|
| 曲风 | 歌的整体感觉 | indie pop=文艺清新 / R&B=慵懒 / folk=民谣 / 古风=中国风 |
| 情绪 | 歌的情绪基调 | 伤感/深情/欢快/梦幻/怀旧 |
| 声线 | 唱歌的人什么嗓音 | 气声女声/甜美女声/温柔男声/低沉男声 |
| BPM | 歌的快慢 | 60=很慢(抒情) / 100=中等 / 140=快(动感) |

## API 密钥获取教程

### 1. Suno Cookie（做歌必须）

Suno 是生成音乐的平台，你需要把登录凭证告诉系统。

1. 打开 Chrome 浏览器，访问 **suno.com**，登录你的账号
2. 按键盘 **F12**（或右键 → 检查），打开开发者工具
3. 点顶部标签栏的 **Application**（应用）
4. 左侧展开 **Cookies** → 点击 `https://suno.com`
5. 在列表里找到 **`__client`** 这一行
6. 双击右边的 Value（值），**Ctrl+A 全选**，**Ctrl+C 复制**
7. 粘贴到设置页面

### 2. Gemini API Key（写歌词用）

Gemini 是 Google 的 AI，用来帮你写歌词。

1. 打开 **aistudio.google.com/apikey**
2. 用 Google 账号登录（没有就注册一个）
3. 点 **Create API Key**（创建密钥）
4. 复制生成的 Key（以 `AIza` 开头的一串字符）
5. 粘贴到设置页面

### 3. 豆包 API Key（封面图，可选）

豆包是字节跳动的 AI 画图工具，用来给歌曲生成封面。**不配也不影响做歌。**

1. 打开 **console.volcengine.com/ark**
2. 注册/登录火山引擎账号
3. 找到「模型推理」→「API Key 管理」
4. 点「创建 API Key」，复制
5. 粘贴到设置页面

## 常见问题

**Q: 做一首歌要多久？**
A: 通常 2-5 分钟。网页上会显示进度。

**Q: 做完的歌在哪？**
A: 默认保存在桌面的「做歌输出」文件夹里。你也可以在网页上改保存位置。

**Q: Suno 报错 422？**
A: 在浏览器打开 suno.com，随便点一下页面，然后回来重试。

**Q: 积分用完了怎么办？**
A: 在设置页面点「查询积分」看余额。用完了需要去 suno.com 充值。

**Q: 可以同时做多首歌吗？**
A: 可以，一首做完再做下一首就行。

**Q: 我想改 API 密钥怎么办？**
A: 打开网页，点左边「⚙️ 设置」，直接改就行。
